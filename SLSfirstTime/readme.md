# Login Credentials Saver Script

This script is designed to be executed from the command line and is intended to enable the browser to save login IDs and passwords for future use. 

## How to Run the Script

To run this script, use the following command:

```bash
tagui firstTime.tag
```

## Steps for Using the Script

1. **Navigate to the SLS login page**

    Visit the SLS login page by using the following URL:

    ```url
    https://vle.learning.moe.edu.sg/login
    ```

2. **User Choice Prompt**

    The script will prompt you to make a choice:

    ```bash
    ask 1 for SLS or 2 for MIMS login
    ```

3. **Enter Login Credentials and Save**

    Please enter one of your login ID and password, and press 'Enter'. When prompted to save the passwords, click 'YES' to allow the browser to use your login credentials in future sessions. Click "OK" to proceed.

4. **Continuing Script Execution**

    The script will continue to execute, and if you wish to quit live mode, simply type 'done'.

> Note: This script is for educational and illustrative purposes. Use it responsibly and in compliance with the website's terms and policies.
